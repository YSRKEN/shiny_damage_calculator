# shiny-damage-calculator

シャニマスの審査員への通常アピールと思い出アピールのダメージを計算するアプリです。

## 開発コマンド

プロジェクトディレクトリで次のコマンドを実行します。

### `npm run dev`

Vite の開発サーバを起動します。既定ではローカル環境でアプリを確認できます。

### `npm run test -- --runInBand`

Jest で既存テストを実行します。

### `npm run build`

本番用ビルドを `build` ディレクトリへ出力します。

### `npm run preview`

Vite のプレビューサーバでビルド済みアプリを確認します。

## 補足

- 静的ファイルは `public` ディレクトリに配置しています。
- エントリ HTML はリポジトリ直下の `index.html` です。
